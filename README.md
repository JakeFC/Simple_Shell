# simple_shell

As Students at Holberton School Tulsa, we built a simple command-line interpreter using C language. Shell running with ./hsh

## Example of Code

$ ./hsh
($) /bin/ls
hsh main.c shell.c
($) exit
$ echo "echo hello" | ./hsh
($) hello
$


## How to Compile

gcc -Wall -Werror -Wextra -pedantic *.c -o hsh
